# 🍅 Tomato Novel (番茄短篇AI创作)

> **Guided Supernova 驱动的沉浸式 AI 短篇小说创作引擎**

Tomato Novel 是一款专为创作者设计的 AI 辅助写作工具。它打破了传统文档编辑器的束缚，引入了**无限画布 (Infinite Canvas)** 与**星源视觉系统 (Guided Supernova Design)**，将 AI 的推演能力与人类的叙事灵感深度融合。

![Vision Interface](https://img.shields.io/badge/Design-Guided--Supernova-ff7afb)
![Tech Stack](https://img.shields.io/badge/Stack-React%20%7C%20FastAPI%20%7C%20SQLite-00e3fd)

## 🌌 核心特性

- **✨ 引导式 AI 巫师**：通过多轮对话流，AI 协助用户从模糊的“核心构思”推演至详细的“全书补全”，自动化生成时代背景、关键道具与结局倾向。
- **📽️ 无限漫游画布**：基于自由平移与缩放的无限画布架构，将故事的“起、承、转、合”转化为可视化的神经元网络，让大纲逻辑一目了然。
- **🎨 沉浸式视觉美学**：全站遵循 Guided Supernova 视觉规范，采用极高饱和度的霓虹色彩、玻璃拟态面板以及流光连线效果，打造科幻感的创作氛围。
- **⚡ 实时创作修订**：支持“时空锚点”快速保存，核心设定一键修订，并能自动分析对后续章节的影响。

## 🛠️ 技术架构

### 前端 (Frontend)
- **Framework**: React 18 + Vite
- **UI Components**: Ant Design (Deeply Customized) + TailwindCSS
- **State Management**: React Hooks & Context API
- **Interaction**: Canvas-based Infinite Pan & Zoom Architecture

### 后端 (Backend)
- **Environment**: Python 3.10+
- **Framework**: FastAPI (High-performance Async)
- **Database**: SQLite (Local Database)
- **AI Integration**: 支持自定义 LLM 接口接入

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/[your-username]/tomato-novel.git
cd tomato-novel
```

### 2. 启动后端 (Python)
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 3. 启动前端 (Node.js)
```bash
cd frontend
npm install
npm run dev
```
